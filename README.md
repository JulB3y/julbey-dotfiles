# julbey-dotfiles
This repo is a collection of my dotfiles. it includes configs for programs i use with my CachyOS Hyprland setup.

## how to use
you're free to use any file and copy it to your paths. install instructions for each program will be added soon. to install all simply execute the install script.

## install script
with the install script you can install all of the provided configs. it simply copies them to the individual folders. make sure to select the prefered config with changing the number in the first line of the script.

## my setup
- distro: [CachyOS](https://cachyos.org/)
- compositor: [hyprland](https://hypr.land/)
- status bar: [waybar](https://github.com/Alexays/Waybar) 
- keyboard remapping: [keyd](https://github.com/rvaiya/keyd)
- note taking: [obsidian](https://obsidian.md/)
